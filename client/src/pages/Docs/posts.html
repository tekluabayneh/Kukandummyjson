posts-Docs

get all posts

By default you will get 24 items, use Limit to paginate through all items. 

 
            fetch('http://localhost:8000/posts?api_key=your_api-key')
            .then(res=>res.json())
            .then(json=>console.log(json))

get single post

get single post by thier id

 
            fetch('http://localhost:8000/posts/getpostById/6?api_key=your_api-key')
            .then(res=>res.json())
            .then(json=>console.log(json))

add new post

please not that if the post id already exist it wont work consider adding new post, if you get post already exist message

 
               fetch('http://localhost:8000/posts/addposts?api_key=your_api-key',{
            method:"POST",
            body:JSON.stringify({
                    id: 'exa-id',
                    title: 'exa-title',
                    body: 'exa-body',
                    author:'exa-author',
                    views: 'exa-454',
                    userId: 'exa-userid',
                })
        })
            .then(res=>res.json())
            .then(json=>console.log(json))

get limited posts

get limited posts amount by just adding limit number into url params

 
            fetch('http://localhost:8000/posts/limit/2?api_key=your_api-key')
            .then(res=>res.json())
            .then(json=>console.log(json))

search posts

you can also search post by views number not that the server will look for grater than the number you give it

 
            fetch('http://localhost:8000/posts/search/1400?api_key=apikey')
            .then(res=>res.json())
            .then(json=>console.log(json))

delete posts

you can also delete post by id not that the actual post data will not be deleted but it will response back with the deleted post data 

 
            fetch('http://localhost:8000/posts/deletepost/4?api_key=apikey')
            .then(res=>res.json())
            .then(json=>console.log(json))

update user

update user providing new data but remember the actual data won`t be updated but instead we send back with the updated new data along with old one

 
         fetch('http://localhost:8000/posts/updatepost?api_key=your_api-key',{
            method:"PUT",
            body:JSON.stringify(
                {
                    id: 'exa-id',
                    title: 'exa-title',
                    body: 'exa-body',
                    author:'exa-author',
                    views: 'exa-454',
                    userId: 'exa-userid',
                }
            )
        })
            .then(res=>res.json())
            .then(json=>console.log(json))
